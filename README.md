# 🍅 Pomodoro Zenith
### Focus Better. Rest Smarter. Stay Consistent.

A modern, distraction-free Pomodoro productivity app designed to help you manage focus sessions, track progress, organize tasks, and build better work habits with ambient sounds and productivity insights.

## Features
- ⏱️ Pomodoro Timer — Focus, short break, and long break sessions
- ✅ Task Manager — Add, track, and complete daily tasks
- 🎵 Ambient Sounds — Rain, Forest, Café, Ocean, Fireplace, Wind, and Lo-fi sounds
- 📊 Productivity Stats — Sessions completed, focus time, tasks, and streak tracking
- 🔔 Notifications — Session alerts and customizable sounds
- ⚙️ Custom Settings — Adjust timer durations and preferences
- 💡 Motivation System — Inspirational quotes and focus reminders
- 🌙 Minimal UI — Clean, calming, and distraction-free design

## Deploy
1. Create a GitHub repository named `pomodoro-zenith`
2. Upload the project files
3. Go to **Settings → Pages**
4. Select the **main** branch and save
5. Live at `https://YOUR_USERNAME.github.io/pomodoro-zenith`

## Stack
**Pure HTML + CSS + Vanilla JavaScript + Local Storage API**  
No frameworks, no dependencies.
